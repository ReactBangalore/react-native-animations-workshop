### Performance Tips
<ul style={{textAlign: "left"}}>
  <Appear>
    <li>ShouldComponentUpdate </li>
    <li>UseNativeDriver = true</li>
    <li>Use interpolation rather than parallal and sequence Animations</li>
    <li>Avoid animating height and width, use scaleX and scaleY instead</li>
    <li>Use this line to make animations work in Android: <br/> <span>"UIManager.setLayoutAnimationEnabledExperimental && UIManager.setLayoutAnimationEnabledExperimental(true)"</span></li>
    <li>Use removeClippedSubview = true (reduce memory)</li>
    <li>Lazy = true, createBottomTabNavigator react navigation</li>
    <li>Avoid to use componentDidUpdate</li>
  </Appear>
</ul>
